sublime text3安装步骤

  1. 下载安装程序,选择合适版本http://www.sublimetext.com/3
  2. 安装package control

    1.访问https://packagecontrol.io/installation
    2.访问完成后打开，按快捷键 ctrl+</font>(数字1键旁边) 3.将步聚2中代码粘贴到，通过<font color=red> ctrl+打开的输出框中，回车等待

    1. 完成上述步骤后，在Preferences菜单下看到Package Settings和Package Control两个菜单了，表示package control已经安装完成；
  3. 点击Package Control,install package ，输入相应关键字，就可以自由安装插件了。
    快捷键：ctrl+shif+p 输入 install package 等待,输入插件名称确定就自动安装
  4. 激活码相信对你来说不是问题^_^

常见问题：

  1. sublime text 3文件名乱码
    在sublime text 3中，Preference, Settings-User，最后加上一行
    "dpi_scale": 1.0,覆盖操作系统设置的DPI。
    这是我的Settings-User

  2. 保证所有UTF-8
    "show_encoding": true,保证右下方状态栏始终显示文件编码格式

  3. 设置html和JavaScript都缩进2格
    tools–HTML/CSS/JS Prettify–set Prettify preferrences，找到html和js区域分别将indent_size设置为2即可;
